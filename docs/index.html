Per-metric outlier-handling method explorer

This is a portfolio demonstration built on synthetic data.

Pick a metric to see its outlier story, the full method comparison, and which method the framework selects. Selection uses only production-measurable signals: predictiveness (does the early reading track the eventual long-term reading) and a calibration gate. A method is disqualified when its early 95 percent CI fails, too often, to cover the eventual long-term estimate, a sign it is over-confident. The last columns are ground-truth checks, shown only to validate the choice, never to make it: RMSE vs truth, and the ship-decision confusion counts (wins shipped, false ships, missed wins, correct holds) for every method.
